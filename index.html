Grid

by Mike Simmonds

Simple, responsive 12 column and micro 4 column CSS grid allowing for rapid prototyping or production. Inspired by Twitter's bootstrap, Zurb's foundation and the 960 grid. View on GitHub or download.

You can also clone the project with Git by running:

$ git clone git://github.com/MikeSimmonds/Grid
